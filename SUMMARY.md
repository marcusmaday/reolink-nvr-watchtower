# Project Summary

Reolink Enhanced API exposes your NVR recordings through a simple API and Home Assistant add-on.

## Main Use Cases

- Search recordings by day, channel, and event type
- Review a searchable timeline of recent events
- Pull playback or download URLs for clips
- Run the service locally or inside Home Assistant
